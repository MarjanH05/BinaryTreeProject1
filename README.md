# BinaryTreeProject1
Binary Tree Project adhering to a series of guidelines outlined by Richard 

Equations should be in the form 5 + 3 (Do not add equal symbol and only allows bidmas operations (+,-,*,/,(),^)

To use the command Line you must set the path to the location of the exe file. It should end in something like this! 
Easiest way is using cd "C:\Users\Marjan Hussain\source\repos\BinaryTree\bin\Debug" (release if using release mode)

To use the command line version write this:

./BinaryTree.exe -s YourEquations.txt -r Output.txt

To use the GUI (preferred version) 
Run the BinaryTreeProjectGUIOwl and select Equation file and do it from there!

Using the tests select the solution and go to properties.
Then, the single start up project should be set to BinaryTreeTests and click apply
Hopefully that is a 100% working test!
